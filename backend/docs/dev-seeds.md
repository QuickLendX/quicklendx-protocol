# Backend Developer Seeds & Playground

## 🚀 Local Seeding
Populate your local database with a sample dataset for development.

### 🏃 Running the Seed
Navigate to the `backend` directory and run:
```bash
npm run seed
```

## 🧪 Testing
Verify the seed script functionality:
```bash
npm run test:seeds
```

## 🛠 API Playground
Use `backend/api-playground.http` with the VS Code "REST Client" extension to test endpoints.

## 🔒 Security
- **No PII**: All data is synthetic.
- **Environment Bound**: Prevents execution in `production`.
