## Recoverability legend

- Recoverable: the caller can fix input, permissions, state selection, or configuration and retry the same business flow.
- Terminal: the requested operation is rejected because the lifecycle state already forbids it; callers should move to another flow instead of retrying blindly.
- Operator: an admin, operator, or deployment maintainer must repair contract configuration, storage, balances, or system state before callers retry.
- Reserved: the ABI error variant exists, but no current production raising site was found in a static source scan.
